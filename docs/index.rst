Rp-Bp: Ribosome profiling with Bayesian predictions
===================================================

An unsupervised Bayesian approach to predict translated open reading frames (ORFs) from ribosome profiles, using an
automatic Bayesian Periodic fragment length and ribosome P-site offset Selection (BPPS).
